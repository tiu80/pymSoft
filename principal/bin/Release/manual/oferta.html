LISTADO DE OFERTAS

En este formulario podemos generar ofertas de acuerdo a nuestras necesidades. En primer lugar debemos seleccionar la lista de donde vamos a obtener los precios (para buscar con la tecla F1). Una vez puesta la lista pasamos a seleccionar los articulos que deseamos ( tambien buscamos con la tecla F1) hasta completar la lista; como vemos a continuacion en la sig. imagen. Tambien podemos poner el periodo de validez de las mismas.

 

 

presionando F2 sobre la grilla de ofertas se nos habilita el campo precio para poder modificar el importe sin necesidad de modificar el precio.(debemos recordar que se modificacion solo sera afectada en la oferta que estamos realizando, no el la lista general)

 

Guarda la lista de ofertas en la base de datos

Carga la lista creada para realizar modificaciones,etc

Borra el items selecionado de la gilla

Imprime la lista de ofertas en la impresora predeterminada

Muestra la vista previa de la lista de ofertas
