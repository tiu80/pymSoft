CLIENTES

En esta ventana tenemos distintas pestañas para completar los datos del cliente. la primera es (General) donde tenemos una grilla con opciones donde podemos elegir entre distintas alternativas; seleccionando cualquiera de ellas nos mostrara la imformacion del ultimo cliente guardado. Tambien tenemos para cargar codigo , nombre , direccion y tipo cliente donde en este pondremos si el cliente es proveedor, gastos o servicios , etc Aca es muy importante definir bien el tipo de cliente.Tambien encontramos las flechas de navegacion para recorrer los registros

En la pestaña Ubicacion tenemos las casillas codigo postal, localidad, provincia, zona, telefono, fax, mail, cabe recordar que con la tecla F1 sobre la casilla podemos abrir un formulario de busqueda por si no recordamos el codigo

En la pestaña Impositivo podemos ver tipo donde elegimos el tipo de encuadre fiscal del cliente: mas abajo encontramos la casilla cuit donde debemos ingresar el numero de inscripcion; este campo es obligatorio y validado siempre y cuando el tipo de cliente sea distinto a consumidor final,en este caso no es obligatorio

En rubro tenemos vendedor donde pondemos el vendedor correspondiente al cliente, lista es la lista de precios en la que vamos a trabajar con ese cliente, rubro y observaciones donde podemos poner un comentario breve de algo particular que queramos aclarar

En estado encontramos fecha de alta donde indicamos cuando se creo el articulo y estado para decir si el producro esta en estado activo o baja

Actualizar : Para actualizar un producto debemos elejir el producto a actualizar (lo hacemos en codigo presionando F1 o el codigo directamente). luego le damos clic al boton actualizar y listo

Borrar: Es el mismo procedimiento que para actualizar pero presionamos el boton borrar donde se abrirar una ventana preguntando si estamos seguros de realizar dicha operacion
