FACTURACION

Podemos observar el menu facturacion donde encontramos la casilla cliente para seleccionar presionando la tecla F1 para buscar o presionando directamente el numero de cliente, tambien encontramos la casilla vendedor,lista , forma de pago para seleccionar si es de contado o cta cte. Mas abajo encontramos codigo articulos; presionado F1 abrimos una ventana de busqueda. A la derecha se encuentra cantidad y descuento/recargo para terminar de cargar el producto en grilla. Aca abajo encontramos los totales donde nos muestra el resultado parcial o total de lo ingresado en la grilla.

MODIFICAR ARTICULO

Presionando la tecla escape sobre codigo producto podemos navegar por la grilla para seleccionar el articulo, presionado enter sobre este nos mostrara la imformacion para poder modificarlo. luego con la tecla insert volvemos a posicionar el cursor sobre codigo producto .

BORRAR ARTICULO

Presionando la tecla escape sobre codigo producto podemos navegar por la grilla. Seleccionamos el articulo a borrar (supr. o delete) y presionamos insert nuevamente para volver al codigo de producto

COMFIRMAR COMPROBANTE

Una vez finalizada la carga de producto presionamos la tecla escape y luego F5, Nos preguntara si deseamos comfirmar el comprobante y dependiendo de la opcion que tengamos configurada se nos abrira una nueva ventana preguntando la forma de pago

En la casilla tipo de pago debemos elegir que opcion de pago elejimos. presionando la tecla F1 podemos ver un menu con las opciones de pago correspondiente. Dependiendo de la opcion seleccionada debemos completar los datos que aparecen. la opciones de pago se cierran con efectivo siempre que el monto total sea distinto a los valores ingresados
